## ⚙️ Installation

### Backend

```bash
pip install -r requirements.txt
```

### Frontend

```bash
npm install
```

---

## ▶️ Run the Project

### Start Backend

```bash
uvicorn main:app --reload
```

### Start Frontend

```bash
npm run dev
```
